flameback-capital/india-market-dataPublic
Cleaned, point-in-time historical data for NSE, BSE, and Indian fixed income.

Pandas-first utilities for fetching, cleaning, and aligning daily OHLCV data for NSE 500 constituents, NIFTY and BSE indices, corporate actions (splits, bonuses, dividends), and a 10-year history of GoI bond yields. Handles index rebalancing and survivorship explicitly.

# typical use from india_market_data import get_universe, get_prices universe = get_universe('NIFTY500', date='2026-04-30') prices = get_prices(universe, start='2018-01-01', adjusted=True)

flameback-capital/backtest-evalPublic
Statistically honest evaluation for systematic strategies.

A backtest evaluation framework with bootstrap confidence intervals on Sharpe, Calmar, drawdown statistics. Includes White’s reality check, deflated Sharpe ratio (Bailey & López de Prado), and combinatorially purged cross-validation. The same framework we use internally.

# confidence intervals on Sharpe from backtest_eval import bootstrap_sharpe ci = bootstrap_sharpe(returns, n_iter=10_000, alpha=0.05) # → Sharpe = 1.42 [0.86, 1.94]

flameback-capital/factor-modelsPublic
Reference implementations of common factor models, adapted for Indian markets.

Fama-French 3 and 5 factor models, Carhart 4-factor, AQR’s QMJ, and several Indian-market-specific factor constructions. Each model ships with a reproduction notebook validating against published results.

Portfolio construction and metrics for systematic strategies.

Equal-weight, inverse-vol, risk-parity, and HRP construction. Performance attribution by sector and factor. Calmar, MAR, Sortino, and a few other ratios computed correctly (including the gotchas in monthly vs daily compounding).

flameback-capital/india-bondsPublic
Fixed-income utilities for the Indian government bond curve.

Curve building (Nelson-Siegel, Svensson), yield/price conversion, accrued interest with day-count conventions used in India, and duration/convexity calculations. Tested against a 10-year CCIL data archive.
