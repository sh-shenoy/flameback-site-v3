v4.6.0 Client app 29 April 2026 Client-facing

Portfolio composition view & monthly statements

A redesigned holdings view in the client app, downloadable monthly statements, and a faster cold-start on the mobile app. The biggest client-facing update since the November redesign.

Added
  • New Composition tab in the client app showing live sector exposure, top-10 holdings with weights, and market-cap distribution per portfolio
  • Downloadable monthly statements in PDF, available on the 5th of every month for the prior month
  • Push notifications for monthly statement availability and material portfolio changes (opt-in)
  • Hindi language toggle in app settings, in addition to existing English
Improved
  • Mobile cold-start time reduced from 2.8s to 1.1s (median across iOS and Android)
  • Portfolio overview now shows TWRR alongside absolute returns for clearer benchmark comparison
  • Tax document section reorganised by financial year
Fixed
  • Edge case where corporate-action adjustments displayed incorrectly for split-adjusted holdings
  • Notification timestamps now respect device timezone

v4.5.2 FIST 18 April 2026 Internal

Kotak Securities integration & reduced slippage

A third broker now lives inside FIST’s smart-order router. Aggregate execution costs down 12% in A/B testing against single-broker baseline. Routing is transparent to strategies.

Added
  • Kotak Securities broker adapter inside FIST — FIX 4.4, full order types supported
  • Per-broker execution analytics dashboard showing fill rates, latency, and effective spread
  • Auto-failover: if a broker’s API drops, FIST routes to next-best available within 200ms
Improved
  • Smart-order router now scores brokers on real-time spread + depth + historical latency (was: spread only)
  • Median slippage reduced from 8.4 bps to 7.4 bps across the strategy book
  • Pre-trade cost estimates now incorporate per-broker historical realisation

v4.5.0 QLab 05 April 2026 Internal

Hosted backtesting on cloud, internal API v2

A complete rewrite of the QLab compute layer. Backtests that took 40 minutes now run in 6. A new internal API gives the research team programmatic access to the data lake and signal evaluation framework.

Added
  • Cloud-hosted backtest workers with parallel execution across signal variants
  • Internal API v2 with endpoints for universe, signals, backtest, and evaluation
  • Signal versioning and lineage tracking — every backtest result is traceable to a specific git commit
  • Notebook templates for common research workflows (factor zoo, regime overlays, turnover analysis)
Improved
  • Median backtest runtime reduced from 38 min to 6 min for an 8-year NSE 500 momentum backtest
  • Memory footprint on local research machines reduced by ~70% (compute now off-machine)
  • Evaluation framework v3 with bootstrap confidence intervals on all summary statistics
Deprecated
  • Internal API v1 endpoints will be removed in v4.7. Migration guide in /docs/api-migration

Tax-aware reporting for PMS portfolios

A new PMS reporting layer that surfaces short-term vs long-term gain composition, harvesting opportunities, and trailing-12-month realised gains in client-friendly format.

v4.4.0 Strategy R&D 28 February 2026 Internal

New strategy evaluation framework

A formalised five-stage evaluation process from signal discovery through deployment, codified into the QLab workflow. Drawing on lessons from 50+ research iterations.

Added
  • Stage gates: discovery → formulation → backtest → paper-trade → production
  • Evaluation criteria documented for each stage with minimum thresholds
  • Mandatory pre-registration of hypotheses before any backtest (eliminates p-hacking)
  • Hexagonal radar scoring system: Performance / Sharpe / Scalability / Correlation / Drawdown / Stability

v4.3.0 Client app · Platform 12 February 2026 Major

Cruise integration: one platform, multiple offerings

The Cruise advisory product has been folded into the main Flameback client app. Clients now have a unified view across Advisory, PMS, and (where applicable) International portfolios in one interface. The Cruise sub-brand is being wound down.

Added
  • Unified portfolio view spanning Advisory, PMS, and International in a single dashboard
  • Cross-portfolio aggregation: total wealth, asset allocation, and currency exposure across products
  • Migration path for existing Cruise users with no action required from clients
Deprecated
  • The Cruise app will redirect to the Flameback app through end of June 2026, then sunset
  • Cruise-branded materials are being retired from client communications
