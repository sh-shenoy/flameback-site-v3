iiiAccount Aggregator

Auto-refreshed, with consent.

The map updates itself through India’s SEBI- and RBI-supervised Account Aggregator framework. Banks, depositories, mutual fund registries, and insurers stream verified data on a consent-based, time-bound, purpose-bound basis. No scraping, no shared passwords.

  • One consent flow links every connected financial account in the household
  • Data refreshes automatically on the cadence you set — daily, weekly, or monthly
  • Read-only by design: the tool cannot transact, transfer, or alter any account
  • You can revoke consent at any time, which stops all data flow immediately

ivLimitations

What this tool does not do.

The Asset Map is a viewing and structuring tool. It does not move money, execute trades, or recommend specific products. Insurance values shown are cover sums, not surrender values. Real estate values rely on the estimates you enter unless connected to a registered valuer feed. The projection uses nominal long-run averages; the actual sequence of returns can vary widely and is not modelled. For decisions involving material capital, we recommend a personal consultation.
