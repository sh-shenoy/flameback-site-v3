How to read this page.

Each tile shows the current value, the change from the prior reported period, and a 12-month trend. The arrow colour reflects the direction the number moved, not whether that is good news — rising inflation reads as “up” but is usually unwelcome. Sources sit below each tile and link to the official series. Refresh cadences differ: weekly for FX reserves, monthly for PMI, CPI, and IIP, quarterly for GDP, on announcement for the repo rate.
